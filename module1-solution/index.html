Lunch Checker Solved

App to check if too much items are inserted:

  • Separate items by ","
    • If item's count less or equal to 3 return "Enjoy!(green)"
    • If item's count more than 4 return "Too much!(green)"
    • IF item's count is zeor return "Please enter data first"(red)
  • Cases "item1, item2,,item3" and "item1, ,item2, item3" are counted as 3 items
